Planning Data Specification Form Generator (Prototype)

Generate web forms and PDFs from the MHCLG planning data specification.

This prototype demonstrates how planning application forms can be generated directly from the MHCLG planning data standards, rather than being manually designed or hard-coded. It shows how the published schemas can drive consistent digital forms, helping suppliers understand how standards can be implemented in practice. While not a production system, it provides a practical example implementation that developers can explore to better understand the structure of the standards, field definitions and validation logic, and how these could be incorporated into future planning software. It is intended as a technical learning resource and a starting point for implementation for the community.

This experimental service could be withdrawn or unavailable without notice.
